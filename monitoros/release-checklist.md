# MonitorOS™ Release Checklist

Before every push:

- [ ] npm install completed
- [ ] npm run dev works
- [ ] npm run build works
- [ ] No node_modules committed
- [ ] No .next folder committed
- [ ] Page renders in Codespaces preview
- [ ] Branding remains consistent
- [ ] README or CTO docs updated
- [ ] Commit message is clear
- [ ] Netlify deploy checked after push
